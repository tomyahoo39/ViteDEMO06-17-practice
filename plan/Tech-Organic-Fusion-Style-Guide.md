# 個人作品集網站風格指南
## Tech-Organic Fusion（科技與有機結合風）

---

# 1. 品牌定位

作為全端開發者，個人作品集網站不只是展示作品的平台，更是個人品牌的延伸。

「科技與有機結合風（Tech-Organic Fusion）」的核心理念在於呈現 **邏輯與感性並存** 的開發者形象。

透過現代化技術架構與柔和手繪元素的融合，展現：

- 嚴謹的程式設計能力
- 良好的系統架構思維
- 對使用者體驗的細膩觀察
- 對設計與美感的重視

此風格傳達的訊息是：

> 我不只是開發功能，而是將技術與體驗共同打造為完整作品。

---

# 2. 設計核心概念

## 邏輯 × 感性

網站視覺應同時具備：

### 理性（Technology）

- 工整的版面系統
- 清晰的資訊層級
- 良好的閱讀體驗
- 高效能前端架構

### 感性（Organic）

- 手繪感邊框
- 自然筆觸線條
- 柔和色彩搭配
- 筆記式視覺元素

兩者並非對立，而是互相襯托。

---

# 3. Design Language 設計語彙

## 3.1 版面系統

### 工整矩陣 + 手繪邊框

使用 CSS Grid 建立理性的版面秩序。

搭配：

- SVG 手繪邊框
- 不規則區塊
- 筆記感標註元素

形成科技與有機共存的視覺語言。

### 建議實作

```css
display: grid;
grid-template-columns: repeat(12, 1fr);
gap: 2rem;
```

---

## 3.2 色彩系統

### 主色調

| 類型 | 顏色 |
|--------|--------|
| 背景色 | #FFFDF5 |
| 文字色 | #333333 |
| 莫蘭迪綠 | 輔助色 |
| 陶土紅 | 強調色 |

### 設計原則

避免高對比科技藍。

採用低飽和色彩，營造：

- 溫暖感
- 專業感
- 藝術感
- 閱讀舒適度

---

## 3.3 字體系統

### 標題字體

Lora

特色：

- 優雅襯線
- 人文氣息
- 藝術感強

### 內文字體

Inter

特色：

- 現代感
- 高可讀性
- 技術文件友善

### 建議組合

```css
font-family: 'Lora', serif;
font-family: 'Inter', sans-serif;
```

---

# 4. 互動設計

## Hover 微互動

當滑鼠移入時：

- 手繪邊框微幅晃動
- SVG 路徑繪製動畫
- 卡片輕微上浮
- 陰影柔和增加

### 動畫感受

模擬鉛筆在紙上描繪的過程。

建議使用：

- GSAP
- SVG Stroke Animation
- Motion One

---

# 5. 專案卡片設計

## 開發者工作日誌概念

每張作品卡片除了展示資訊外，額外加入：

### 手繪筆記區

例如：

- API 效能優化
- SEO 提升
- 系統架構重構
- 使用者體驗改善

呈現效果像開發過程中的工作筆記。

### 範例標註

```text
✏ 解決 SSR SEO 問題

✏ 首屏載入速度提升 45%

✏ JWT 權限系統重構
```

讓作品更具故事性與技術深度。

---

# 6. 效能優化策略

## SVG 優先

裝飾元素建議全部採用 SVG：

- 檔案小
- 可動畫化
- 可縮放
- SEO 友善

避免：

- 大量 PNG
- 大量裝飾圖片

---

## 不規則形狀

建議使用：

```css
clip-path
mask-image
```

實現自然手繪輪廓。

---

## 圖片優化

- WebP
- Lazy Loading
- Responsive Images
- Nuxt Image Module

---

# 7. 留白策略（White Space）

此風格成功的關鍵並非更多裝飾。

而是讓裝飾與內容保持平衡。

設計原則：

- 大量留白
- 區塊間距充足
- 每個視覺焦點皆有呼吸空間

避免：

- 過多手繪元素
- 過度堆疊裝飾
- 過密集內容
