# 個人作品集網站規劃書

## 1. 品牌定位與視覺設計
* **品牌風格**：手繪藝術風 (Hand-drawn Artistic Style)
* **色調策略**：
    * **背景色**：奶油白 (`#FFFDF5`)，打造溫暖手感。
    * **輔助色**：莫蘭迪綠、陶土紅（低飽和度）。
    * **文字色**：深灰色 (`#333333`)，確保易讀性。
* **排版建議**：
    * **元素處理**：使用不規則手繪邊框 (CSS `border-image` 或 SVG)。
    * **字體選擇**：標題採用手寫感襯線體（如 `Lora`），正文使用清晰無襯線體（如 `Inter`）。

## 2. SEO 架構與關鍵字策略
* **網站結構 (Sitemap)**：
    * `/`：首頁（價值主張）
    * `/projects`：作品列表
    * `/projects/[slug]`：單一作品詳情
    * `/about`：關於我
    * `/contact`：聯絡頁面
* **核心策略**：
    * 每個頁面設置獨立的 Meta Title 與 Description。
    * 使用 Schema.org 結構化資料 (`SoftwareApplication`)。
    * URL 使用描述性關鍵字。

## 3. 核心頁面內容規劃
* **首頁**：明確的價值主張（例如：「用藝術的思維，寫出結構嚴謹的程式碼」）。
* **作品詳情頁 (STAR 架構)**：
    * **背景 (Situation)**：解決的問題與痛點。
    * **技術棧 (Stack)**：Vue, Node.js 等核心工具。
    * **解決方案 (Action)**：架構設計與技術克服過程。
    * **結果 (Result)**：量化數據（效能提升、轉換率等）。
* **關於我**：強調學習曲線與實作熱情。
* **聯絡頁面**：精簡化表單，提供 Email 一鍵複製功能。

## 4. 技術與效能建議
* **技術棧**：Nuxt.js (SEO 友善) + Tailwind CSS (輕量靈活)。
* **效能優化**：
    * 靜態資料管理：使用 `projects.json` 作為資料驅動來源。
    * 圖片優化：自動化格式轉換 (WebP)。
    * 資源管理：所有手繪裝飾元素採用 SVG 壓縮格式。

## 5. 微互動 (Micro-interactions) 建議
1.  **手繪邊框 Hover**：卡片邊框呈現微弱的抖動效果。
2.  **卷軸動畫**：使用 GSAP 實現鉛筆繪圖式的出現動畫。
3.  **點擊反饋**：按鈕按下的筆觸加粗效果。
